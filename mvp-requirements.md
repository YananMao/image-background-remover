# Image Background Remover - MVP 需求文档

## 1. 产品定位

一个简洁的在线抠图工具，用户上传图片后一键去除背景，下载透明PNG。

## 2. 目标用户

- 电商卖家（商品图抠底）
- 自媒体创作者（头像、素材处理）
- 普通用户（证件照换底、简单P图）

## 3. 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽/点击上传，限制格式 JPG/PNG/WebP，最大 10MB | P0 |
| 背景移除 | 调用 remove.bg API，返回透明背景 PNG | P0 |
| 预览对比 | 原图 vs 抠图效果，支持缩放查看 | P0 |
| 下载结果 | 一键下载透明 PNG | P0 |
| 加载状态 | 上传/处理中显示 loading 动画 | P0 |
| 错误提示 | 上传失败、API 限额、网络异常等友好提示 | P0 |

## 4. 非功能需求

| 项目 | 要求 |
|------|------|
| 性能 | 单张图片处理 < 10s（取决于 remove.bg 响应） |
| 部署 | Cloudflare Pages + Workers |
| 存储 | 无持久化存储，全内存处理 |
| 安全 | API Key 通过环境变量注入，不暴露前端 |
| 兼容 | 支持 Chrome/Firefox/Safari/Edge 最新版 |
| 移动端 | 响应式布局，手机可用 |

## 5. 技术架构

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────┐
│  Cloudflare     │     │  Cloudflare      │     │  remove.bg  │
│  Pages (前端)   │────▶│  Workers (API)   │────▶│  API        │
│  React/Next.js  │◀────│  Hono 框架       │◀────│             │
└─────────────────┘     └──────────────────┘     └─────────────┘
```

### 5.1 技术栈详情

**前端**
- 框架：Next.js 14+ (App Router)
- 样式：Tailwind CSS
- 上传组件：React Dropzone
- 部署：Cloudflare Pages

**后端**
- 运行时：Cloudflare Workers
- 框架：Hono
- 部署：Cloudflare Workers

**第三方服务**
- 抠图 API：remove.bg

## 6. 用户流程

```
上传图片 → 显示预览 → 点击"去除背景" → Loading → 显示对比 → 下载PNG
```

## 8. API 接口设计

### 8.1 去除背景

**请求**
```
POST /api/remove-background
Content-Type: multipart/form-data

file: <image binary>
```

**响应**
```
Content-Type: image/png

<transparent png binary>
```

**错误响应**
```json
{
  "error": "错误类型",
  "message": "详细错误信息"
}
```
